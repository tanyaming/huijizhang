## 系统概述

### 什么是 JinBooks？

JinBooks 是一款功能齐全的财务记账系统，专为中小企业设计，帮助财务团队告别繁琐的 Excel 手工记账，实现财务管理的数字化、自动化和智能化。

### 核心优势

✅ **无纸化办公** - 全流程电子化管理，提高工作效率  
✅ **自动化处理** - 月末自动结转、报表自动生成  
✅ **多账套支持** - 支持无限账套，适合集团化管理  
✅ **智能分析** - 仪表盘可视化，财务状况一目了然  
✅ **规范管理** - 符合会计准则，确保账务准确性  

### 系统功能模块

| 功能模块 | 主要功能 | 适用人员 |
|---------|---------|---------|
| 账套管理 | 创建、切换、管理多个账套 | 财务主管 |
| 科目管理 | 会计科目设置、辅助核算配置 | 财务主管、会计 |
| 凭证管理 | 录入、审核、整理凭证 | 会计、出纳 |
| 薪资管理 | 员工薪资计算、工资凭证生成 | 人事、会计 |
| 日记账 | 现金、银行日记账管理 | 出纳 |
| 月末结账 | 自动结转损益、生成报表 | 财务主管 |
| 财务报表 | 资产负债表、利润表、现金流量表等 | 全体财务人员 |
| 仪表盘 | 财务数据可视化分析 | 管理层、财务主管 |

## 系统登录与账户管理

### 首次登录

1. **访问系统**
   - 打开浏览器，输入系统地址（由IT部门提供）
   - 推荐使用 Chrome、Edge 或 Firefox 浏览器

2. **输入登录信息**
   - 用户名：由管理员分配
   - 密码：首次登录使用初始密码，登录后请立即修改

3. **修改初始密码**
   - 点击右上角用户头像 → 个人中心
   - 选择"修改密码"
   - 输入旧密码和新密码（建议使用字母+数字+符号组合）

### 权限说明

系统根据岗位分配不同权限：

- **财务主管**：全部功能权限
- **会计**：凭证录入、报表查看
- **出纳**：日记账管理、凭证录入
- **查看人员**：仅查看报表和数据

## 账套管理

### 什么是账套？

账套是一个独立的账务体系，包含完整的科目、凭证、报表等数据。一个企业可以创建多个账套，例如：
- 不同年度的账套（2024年账套、2025年账套）
- 不同公司的账套（母公司账套、子公司账套）
- 不同业务的账套（主营业务账套、项目账套）

### 创建新账套

1. **进入账套管理**
   - 导航栏：系统管理 → 账套管理
   - 点击"新增账套"按钮

2. **填写账套信息**
   ```
   账套名称：2025年度账套
   会计期间：2025-01-01 至 2025-12-31
   企业名称：XX有限公司
   纳税人识别号：91XXXXXXXXXXXXXXXXX
   企业类型：一般纳税人/小规模纳税人
   行业类型：制造业/服务业/商贸业等
   ```

3. **初始化科目**
   - 系统会根据行业类型自动生成标准科目
   - 可以在科目管理中进行调整

4. **设置期初余额**
   - 在"期初余额"模块录入各科目的期初数据
   - 确保借贷平衡后才能开始记账

### 切换账套

- 点击顶部导航栏的账套名称
- 从下拉列表中选择要切换的账套
- 系统会自动加载该账套的数据

### 设置默认账套

- 在账套列表中，点击"设为默认"
- 下次登录时自动进入该账套

## 科目管理

### 科目体系

JinBooks 支持六大类会计科目：

| 科目类别 | 编码范围 | 余额方向 | 示例 |
|---------|---------|---------|------|
| 资产类 | 1001-1999 | 借方 | 库存现金、银行存款、应收账款 |
| 负债类 | 2001-2999 | 贷方 | 短期借款、应付账款、应交税费 |
| 共同类 | 3001-3999 | 双向 | 衍生工具 |
| 所有者权益类 | 4001-4999 | 贷方 | 实收资本、资本公积、未分配利润 |
| 成本类 | 5001-5999 | 借方 | 生产成本、制造费用 |
| 损益类 | 6001-6999 | 双向 | 主营业务收入、管理费用 |

### 新增科目

1. **进入科目管理**
   - 导航栏：基础设置 → 科目管理

2. **添加一级科目**
   ```
   科目编码：6401
   科目名称：销售费用
   科目类别：损益类
   余额方向：借方
   ```

3. **添加明细科目**
   ```
   上级科目：6401 销售费用
   科目编码：640101
   科目名称：广告费
   ```

### 辅助核算设置

辅助核算用于更精细的财务管理：

- **客户往来**：应收账款、预收账款
- **供应商往来**：应付账款、预付账款
- **部门核算**：费用类科目
- **项目核算**：成本类科目
- **存货核算**：原材料、库存商品

**设置方法：**
1. 选择科目，点击"编辑"
2. 勾选需要的辅助核算类型
3. 保存后，在凭证录入时会要求选择辅助核算项

## 凭证管理

### 凭证类型

- **记账凭证**：通用凭证（记）
- **收款凭证**：涉及现金、银行存款增加（收）
- **付款凭证**：涉及现金、银行存款减少（付）
- **转账凭证**：不涉及现金、银行存款（转）

### 录入凭证

1. **进入凭证录入**
   - 导航栏：凭证管理 → 凭证录入
   - 点击"新增凭证"

2. **填写凭证信息**
   ```
   凭证日期：2025-02-13
   凭证字号：记-001（系统自动生成）
   附件张数：3
   ```

3. **录入分录**
   
   **示例：采购原材料**
   ```
   借：原材料 - A材料          10,000.00
       应交税费 - 应交增值税      1,300.00
   贷：银行存款 - 工商银行       11,300.00
   ```

4. **辅助核算录入**
   - 如果科目设置了辅助核算，需要选择对应的辅助项
   - 例如：应收账款需要选择客户名称

5. **借贷平衡检查**
   - 系统自动计算借贷差额
   - 只有借贷平衡才能保存凭证

### 凭证模板

为提高效率，可以创建常用凭证模板：

1. **创建模板**
   - 凭证管理 → 凭证模板
   - 录入常用业务的凭证分录
   - 保存为模板

2. **使用模板**
   - 新增凭证时，点击"选择模板"
   - 选择对应的业务模板
   - 修改金额和日期即可

**常用模板示例：**
- 销售商品收款
- 采购付款
- 计提工资
- 发放工资
- 缴纳社保
- 结转成本

### 凭证审核

1. **审核流程**
   - 会计录入凭证 → 财务主管审核
   - 审核通过后凭证才能记账

2. **审核操作**
   - 凭证管理 → 凭证审核
   - 查看凭证详情
   - 点击"审核通过"或"退回修改"

### 凭证整理

月末可以对凭证进行整理：

- **重新编号**：按日期顺序重新排列凭证号
- **删除空白凭证**：清理未使用的凭证
- **凭证打印**：批量打印凭证

## 薪资管理

### 员工信息管理

1. **添加员工**
   - 人事管理 → 员工管理
   - 录入员工基本信息：姓名、工号、部门、岗位

2. **设置薪资结构**
   ```
   基本工资：5,000
   岗位津贴：1,000
   绩效工资：2,000
   社保个人部分：-500
   公积金个人部分：-600
   个人所得税：-XXX（系统自动计算）
   ```

### 薪资计算

1. **进入薪资管理**
   - 薪资管理 → 月度薪资

2. **生成薪资表**
   - 选择月份
   - 点击"生成薪资"
   - 系统自动计算应发工资、扣款、实发工资

3. **个税计算**
   - 系统根据最新个税政策自动计算
   - 支持专项附加扣除设置

### 生成工资凭证

1. **计提工资凭证**
   ```
   借：管理费用 - 工资          50,000
       销售费用 - 工资          30,000
   贷：应付职工薪酬 - 工资       80,000
   ```

2. **发放工资凭证**
   ```
   借：应付职工薪酬 - 工资       80,000
   贷：应付职工薪酬 - 社保        5,000
       应付职工薪酬 - 公积金      6,000
       应交税费 - 个人所得税      3,000
       银行存款                  66,000
   ```

## 日记账管理

### 现金日记账

1. **录入现金收支**
   - 日记账管理 → 现金日记账
   - 记录每笔现金收入和支出

2. **日记账格式**
   ```
   日期：2025-02-13
   凭证号：收-001
   摘要：收到销售款
   收入：5,000
   支出：-
   余额：15,000
   ```

### 银行日记账

1. **账户管理**
   - 添加银行账户：工商银行、建设银行等
   - 设置账户初始余额

2. **录入银行收支**
   - 根据银行回单录入
   - 与凭证关联

3. **银行对账**
   - 月末与银行对账单核对
   - 编制银行存款余额调节表

### 月末汇总

- 系统自动汇总当月现金、银行收支
- 生成日记账汇总表
- 与总账核对一致

## 月末结账

### 结账前准备

1. **检查凭证**
   - 所有凭证已审核
   - 借贷平衡
   - 无未记账凭证

2. **核对账务**
   - 银行存款与银行对账单一致
   - 现金账实相符
   - 往来账核对清楚

3. **计提费用**
   - 计提折旧
   - 计提工资
   - 计提税金

### 月末结转

1. **进入月末处理**
   - 月末管理 → 月末结转

2. **自动结转损益**
   - 系统自动生成结转凭证
   - 将收入、费用结转至本年利润

   **结转凭证示例：**
   ```
   借：主营业务收入            500,000
       其他业务收入             50,000
   贷：本年利润                550,000

   借：本年利润                400,000
   贷：主营业务成本            300,000
       管理费用                 50,000
       销售费用                 30,000
       财务费用                 20,000
   ```

3. **生成报表**
   - 结转完成后，系统自动生成财务报表
   - 资产负债表
   - 利润表
   - 现金流量表
   - 科目余额表

### 结账操作

1. **执行结账**
   - 月末管理 → 结账
   - 确认无误后点击"结账"

2. **结账后限制**
   - 已结账月份不能再录入凭证
   - 如需修改，必须先反结账

3. **反结账**
   - 仅财务主管有权限
   - 谨慎操作，会影响后续月份数据

## 财务报表

### 资产负债表

**功能说明：**
- 反映企业某一时点的财务状况
- 展示资产、负债、所有者权益

**查看方式：**
1. 报表管理 → 资产负债表
2. 选择会计期间
3. 点击"生成报表"

**报表结构：**
```
资产 = 负债 + 所有者权益

资产：
  流动资产：货币资金、应收账款、存货等
  非流动资产：固定资产、无形资产等

负债：
  流动负债：短期借款、应付账款等
  非流动负债：长期借款等

所有者权益：
  实收资本、资本公积、未分配利润等
```

### 利润表

**功能说明：**
- 反映企业一定期间的经营成果
- 展示收入、成本、费用、利润

**报表结构：**
```
一、营业收入
减：营业成本
    税金及附加
    销售费用
    管理费用
    财务费用
加：其他收益
    投资收益
二、营业利润
加：营业外收入
减：营业外支出
三、利润总额
减：所得税费用
四、净利润
```

### 现金流量表

**功能说明：**
- 反映企业现金流入流出情况
- 分为经营活动、投资活动、筹资活动

**配置方法：**
1. 报表管理 → 现金流量表配置
2. 设置科目与现金流量项目的对应关系
3. 系统自动生成现金流量表

### 科目余额表

**功能说明：**
- 展示所有科目的期初、本期发生、期末余额
- 是编制报表的基础

**查询方式：**
1. 报表管理 → 科目余额表
2. 选择会计期间
3. 可按科目级次筛选

**表格内容：**
```
科目编码 | 科目名称 | 期初余额 | 借方发生 | 贷方发生 | 期末余额
```

### 报表导出

所有报表支持导出：
- Excel 格式
- PDF 格式
- 打印输出

## 常见问题与解决方案

### Q1: 凭证录入时提示"借贷不平衡"？

**原因：** 借方金额合计 ≠ 贷方金额合计

**解决：**
- 检查每笔分录的金额
- 使用系统的"借贷差额"提示
- 确保小数点位数正确

### Q2: 无法结账，提示"存在未审核凭证"？

**原因：** 当月有凭证未经审核

**解决：**
1. 凭证管理 → 凭证审核
2. 审核所有待审核凭证
3. 或删除不需要的凭证

### Q3: 报表数据不正确？

**原因：** 可能是科目余额或结转有误

**解决：**
1. 检查科目余额表是否平衡
2. 检查损益结转凭证是否正确
3. 重新生成报表
4. 如仍有问题，联系技术支持

### Q4: 如何修改已结账月份的凭证？

**解决：**
1. 必须先反结账（需要财务主管权限）
2. 修改凭证
3. 重新结账

**注意：** 反结账会影响后续月份，需谨慎操作

### Q5: 忘记登录密码怎么办？

**解决：**
- 联系系统管理员重置密码
- 或使用"忘记密码"功能（如已配置）

### Q6: 辅助核算项如何添加？

**解决：**
1. 基础设置 → 辅助核算
2. 选择类型（客户、供应商、部门等）
3. 点击"新增"，录入信息

### Q7: 如何批量导入凭证？

**解决：**
1. 凭证管理 → 凭证导入
2. 下载导入模板
3. 按模板格式填写数据
4. 上传文件导入

### Q8: 报表导出后格式错乱？

**解决：**
- 使用 Excel 2016 及以上版本打开
- 或选择 PDF 格式导出
- 检查浏览器兼容性

## 最佳实践建议

### 日常操作规范

1. **及时录入凭证**
   - 每日业务当日录入
   - 避免月末集中录入

2. **规范摘要填写**
   - 摘要清晰明了
   - 便于日后查询

3. **附件管理**
   - 凭证附件张数准确
   - 原始单据妥善保管

4. **定期备份**
   - 每月结账后备份数据
   - 重要操作前备份

### 月末工作流程

```
第1-25日：日常凭证录入
第26-28日：凭证审核、账务核对
第29日：计提费用、月末结转
第30日：生成报表、结账
次月1-3日：报表分析、上报
```

### 权限管理建议

- **最小权限原则**：只分配必要的权限
- **职责分离**：录入与审核分开
- **定期审查**：定期检查权限设置

### 数据安全
